# ai_interview

### 면접을 경험할 수 있는 웹 서비스 입니다.
#### 해당 서비스는 Gemini API 를 사용하여 가상 면접 시스템을 제공하는 것을 목표로 가지고 있습니다.
#### 파일 내 API_KEY 는 유효하지 않은 키 입니다.
#### 만약 해당 서비스를 사용하고 싶은 경우 https://aistudio.google.com/app/apikey?hl=ko 에 접속하여 키를 생성하시면 됩니다.
